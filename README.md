# Sistine's Blog

这是我的个人博客，基于 [Astro](https://astro.build) 与 [Firefly](https://github.com/CuteLeaf/Firefly) 主题构建。

这里会记录一些学习笔记、技术分享和生活随笔。

## 本地运行

```bash
pnpm install
pnpm dev
```

然后打开 http://localhost:4321 即可预览。

## 构建

```bash
pnpm build
```

构建产物输出到 `./dist/` 目录。

## 主题

- 主题：[Firefly](https://github.com/CuteLeaf/Firefly)
- 框架：[Astro](https://astro.build)
